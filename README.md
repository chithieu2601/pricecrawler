This is the project buils a system that searches for similar products and crawls competitor shops from shopify to get:
•	Product availability
•	Price
•	Shop link
The data is used to compare pricing for adjust selling price of Mystic perfume, which helps increase competitive advantage

How to run
On command prompt, move to the file directory, then run server.js (node server.js)
Then, to get data output: localhost:3000/price-check?product=productname (Change productname to the product you’re searching for)
